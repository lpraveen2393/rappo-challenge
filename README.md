# Web Scraper Project

## Overview
A scraper to extract champion information from vendor case study pages and save it in JSON format.

## Folder Structure
```
mainfolder/
│── main.py
│── python_files/
│── scrapper/
│   ├── scraper.js
```

## Dependencies
### Python
Install with:
```sh
pip install -r requirements.txt
```
### Node.js
```sh
cd scrapper
npm install
```

## Usage
Run:
```sh
python main.py
```

## Output
Data is saved in JSON format.
